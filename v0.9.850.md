# v0.9.850

1. **Selection copy fixes** — copying text across output from Claude Code or other TUIs that redraw mid-selection no longer loses the head, tail, or middle of what you highlighted. Dragging the selection shorter (pulling the tail back) also behaves correctly now instead of snapping to the longer pre-shrink text.
2. **Wrapped URLs** — Ctrl+click on a URL that wraps across two lines opens the full URL instead of just the first-line fragment.
3. **Editor (micro) integration** — Save and Quit pill buttons live on the editor overlay alongside the top/bottom jump arrows while you're in micro. Files opened via `yaw filename` from the command line now show up in Recent Files. Editor features trigger reliably even on older installs (termtitle is now enforced).
4. **Terminal reflow** — shrinking a pane (window resize or dragging the split divider) no longer leaves ghost characters past the new right edge that made the pane look like it had a second skinny pane stuck onto it.
5. **Font Family dropdown** — removed the "Custom…" option that silently did nothing when picked. Any previously-set custom font value is migrated to the platform default automatically.
